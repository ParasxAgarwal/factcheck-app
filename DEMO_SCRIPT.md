# 30-second demo script (recording)

1) Open the app home page (show the upload UI).
2) Upload a sample PDF (2–4 pages with some numeric statements).
3) Narrate: “We extract checkable claims (numbers/dates) and verify via live web search (SearXNG).”
4) Show progress bar (“Extracting…” → “Verifying…”).
5) Scroll the Results table:
   - Point at one ✅ Verified row and open the evidence link in a new tab.
   - Point at one ⚠️ Inaccurate row (conflicting numbers) and one ❌ False row (no evidence).
6) End on the table and say: “That’s claim-level verification from a PDF with sources.”

Suggested screen recorder: Loom / OBS / macOS QuickTime / Windows Xbox Game Bar.
