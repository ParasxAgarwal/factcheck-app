📋 PROJECT OVERVIEW

A professional fact-checking application that analyzes text from PDF documents, extracts claims, and verifies them against multiple search providers. Built with React, TypeScript, and deployed on Cloudflare infrastructure.

🚀 DEPLOYED APPLICATION LINK

▶ Live Demo: https://d99b3c1e.cogculture-factcheck-web.pages.dev

FEATURES:
• PDF upload with intelligent text extraction
• Automated claim extraction and segmentation
• Multi-source claim verification (SearXNG, Brave Search, Firecrawl)
• Professional results display with verdicts (True/False/Unclear)
• Responsive UI with GitHub integration
• Fully accessible and mobile-friendly

HOW TO USE:
1. Visit: https://d99b3c1e.cogculture-factcheck-web.pages.dev
2. Upload a PDF file
3. View extracted claims
4. Check verification results with sources
5. Click sources to verify independently

$ npm install

Build & Deploy:

$ wrangler pages deploy dist/

Testing:
